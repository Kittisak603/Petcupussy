เปรียบเทียบ.container vs .container-fluid

container

กำหนดค่า Property เพิ่เติมในการจัดการ Style โดยไม่ใช้ clss จาก bootstrap

  • background-color
  • text-align
  • padding
  • margin
  • color

container-fluid

กำหนดค่า Property เพิ่เติมในการจัดการ Style โดยไม่ใช้ clss จาก bootstrap

  • background-color
  • text-align
  • padding pt-* Padding-Top (ด้านบน) pb-* Padding-Bottom (ด้านล่าง) ps-* Padding-Start (ซ้ายใน LTR, ขวาใน RTL) pe-* Padding-End (ขวาใน LTR, ซ้ายใน RTL) px-* Padding ซ้าย + ขวา py-* Padding บน + ล่าง
  • margin m-0 ไม่มี margin m-1 Margin เล็กสุด m-2 Margin ขนาดเล็กขึ้น m-3 Margin ปานกลาง m-4 Margin ใหญ่ขึ้น m-5 Margin ใหญ่สุด m-auto กำหนด margin อัตโนมัติ (ใช้สำหรับจัดกึ่งกลาง) ใช้ mt-*, mb-*, ms-*, me-*, mx-*, my-* เพื่อกำหนด margin เฉพาะด้าน
  • color
